## Features
1. User Auth
2. Events (CRUD)
3. Google Maps API Integration
4. User Profiles
5. Discover & Search Events
### Bonus Features
6. Social Metrics & Stats for Events
7. Friends & DM's
8. Events Posts
<br>


## Tech Stack
- MongoDB
- ExpressJS
- React.js
- Node.js
